Usage Examples

Basic Usage:

showSwiftSuccess('Success!', 'Your action was successful.');
showSwiftError('Error', 'Something went wrong.');
showSwiftWarning('Warning', 'Please check your input.');
showSwiftInfo('Info', 'Here is some information.');

Confirmation:

showSwiftConfirm(
    'Delete?',
    'Are you sure?',
    function() { /* on confirm */ },
    function() { /* on cancel */ }
);

Advanced Usage:

swiftAlert.show({
    title: 'Custom Alert',
    message: 'This is a custom message',
    type: 'info',
    buttons: [
        { text: 'Cancel', action: null },
        { text: 'OK', action: function() { console.log('OK clicked'); } }
    ]
});
